# AI-Powered Cybersecurity & IT Helpdesk Automation Platform

## Project Overview

An AI-powered platform that automates IT support and cybersecurity operations through:

- Password Reset Assistant
- Security Awareness Training Bot
- Incident Reporting Assistant
- Internal IT Support Voice Agent
- AI SOC Notification Assistant

## Tech Stack

Frontend:
- React
- Axios

Backend:
- FastAPI
- SQLAlchemy

Database:
- PostgreSQL

AI:
- Google Gemini API

## Running Frontend

cd frontend
npm install
npm run dev

## Running Backend

cd backend
source venv/bin/activate
uvicorn app:app --reload
